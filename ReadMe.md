OhMyDays(오마이 데이즈)


    ▷ 개발 목적 : 현대인에게 힐링의 시간을 제공하고 자신의 하루를 돌아볼 수 있는 그림 일기 앱
    
    ▷ 개발 환경 : 
    
        - 안드로이드 스튜디오
        - Volley 라이브러리를 통한 http 통신
        - SQlite
        - phpMyAdming + Mysql
    
    ▷ 팀 내 역할 :
    
        - 회원 및 일기 DB 명세 및 구현
        - 로그인 및 회원가입 구현
        - 일기 내용 불러오기 기능 구현
        - 로딩 화면 구현
        
     ▷ 한계점 : 
     
        -Grid 형식으로 구현하고 싶었으나 recyclerView를 활용한 GridLayoutManager 사용에 어려움을 느껴 LinearLayout으로 구현하였음.
        -검색 기능 구현 실패
